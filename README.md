# FIFA World Cup Dashboard

An interactive Power BI dashboard project created to analyze FIFA World Cup statistics, attendance trends, winners, and hosting countries from 1930–2014.

## Features
- Attendance analysis by country and year
- Competition statistics visualization
- World Cup winners analysis
- Countries qualified for the tournament
- Host country insights
- Interactive charts and filters

## Tools Used
- Power BI
- Excel
- Data Visualization

## Dashboard Insights
- Identified countries with highest World Cup participation
- Analyzed attendance growth across tournaments
- Visualized winning trends and hosting patterns
- Compared yearly competition statistics
